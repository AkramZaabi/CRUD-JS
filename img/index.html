Projet Web semestriel 2.0

Le projet CRUD de Gestion des étudiants est une application web développée en utilisant JavaScript (JS) qui permet de gérer facilement les informations des étudiants. Avec cette application, les utilisateurs peuvent créer de nouveaux profils étudiants, afficher les détails des étudiants existants, mettre à jour les informations et supprimer des profils d'étudiants.
